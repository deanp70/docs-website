## 통사론

```
NewRelic.withLaunchActivityName(string $activityName)
```

콜드 타임과 핫 타임을 계산하기 위해 앱 실행 시간에 대한 대상 활동 이름을 설정합니다.

## 요구 사항

에이전트 버전 6.9.0 이상.

## 설명

기본적으로 New Relic Android 에이전트는 기본 활동이 로드될 때 콜드 타임과 핫 타임을 계산합니다. 이 API를 사용하면 콜드 타임과 핫 타임을 계산할 때 대상 시작 활동 이름을 사용자 지정할 수 있습니다.

이 API 사용에 대한 일반적인 정보는 [Android SDK API 사용 가이드](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) 를 참조하세요.
