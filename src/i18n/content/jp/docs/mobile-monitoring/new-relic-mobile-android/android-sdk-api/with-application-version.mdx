## 構文

```
NewRelic.withApplicationVersion(string $appVersion)
```

インタラクションに関連付けるアプリケーションのバージョンを設定します。

## 要件

Agentのバージョン4.244.0以上。

## 説明

デフォルトでは、New Relic Android エージェントは`AndroidManifest.xml`または`build.gradle`からアプリのバージョンを取得します。このメソッドを使用すると、その値をオーバーライドし、アプリのバージョンを渡すことでアプリのバージョンを設定できます。アプリのバージョンは New Relic UI のドロップダウンとして表示され、それを使用してインタラクションを並べ替えることができます。

本APIの一般的な使用方法については、 [Android SDK API usage guide](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) をご参照ください。
